    Loads an indexed assets data into memory so it is not loaded from the filesystem. Only use this for preloading as it DOES NOT LOAD THE ASSET, it only stores the raw data for it to be quicker accesible!

    def load(name : String, asset_type : T.class) forall T #

    Loads an Asset. See details. You must provide the correct Asset Type for it to work (Or use the guessing load function). Accepted types are all in Alias Asset. Must be called before using the Asset. Uses cached data if available. Implicitly caches data from zip files because Assets cannot be created from zip entries (this is not recommended). Loads from indexed zip file entries before indexed folders!
